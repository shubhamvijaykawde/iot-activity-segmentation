# Dataset

This project uses the public Industrial IoT dataset available at:

https://zenodo.org/records/15773369

## Download Instructions

1. Download the dataset from the link above
2. Extract the files
3. Navigate to : data-radiant-eval-paper\factory\evaluation\iot_logs
You will find **5 JSONL files** containing IoT sensor logs from industrial machinery.
4. In the code we use this location :- data\data-radiant-eval-paper\factory\evaluation\iot_logs

## Note

The dataset is not included in this repository due to size and licensing constraints.
